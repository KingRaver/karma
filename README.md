## ğŸ—ï¸ Quick Start

### Prerequisites
- Python 3.11 or higher
- macOS (M4 optimized), Linux, or Windows
- API keys from supported providers (see Configuration below)

### Installation

```bash
# Clone and setup
git clone https://github.com/KingRaver/karma.git
cd karma

# Create and activate virtual environment
python3 -m venv venv
source venv/bin/activate  # On macOS/Linux
# OR on Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# For Apple Silicon (M4 MacBook), TensorFlow for macOS is already in requirements
# This provides hardware acceleration for ML models
```

### Configuration

Create a `.env` file in the project root with your API keys:

```bash
# Copy example if available, or create new
cp .env.example .env  # if example exists
# OR
touch .env

# Edit .env with required settings:
```

**Required API Keys:**
```env
# CoinGecko (Required for market data)
COINGECKO_API_KEY=your_coingecko_api_key_here

# AI Providers (At least one required)
ANTHROPIC_API_KEY=your_claude_api_key_here
OPENAI_API_KEY=your_openai_api_key_here
MISTRAL_API_KEY=your_mistral_api_key_here
GROQ_API_KEY=your_groq_api_key_here
```

**Optional Configuration:**
```env
# Social Media Automation (Optional)
TWITTER_USERNAME=your_username
TWITTER_PASSWORD=your_password

# Google Sheets Export (Optional)
GOOGLE_SHEETS_CREDENTIALS_PATH=path/to/credentials.json
GOOGLE_SHEETS_SPREADSHEET_ID=your_spreadsheet_id

# Database (Optional - defaults to data/crypto_history.db)
DATABASE_PATH=data/crypto_history.db
```

### Database Initialization

```bash
# Initialize the database
python src/database.py

# This creates:
# - data/crypto_history.db (main database)
# - data/backup/ (backup directory)
```

### Launch

```bash
# Run the main bot
python src/bot.py

# Or test specific components
python src/network_test.py  # Test API connectivity
python src/test_browser.py  # Test browser automation
python tech_test.py         # Run technical tests
```
